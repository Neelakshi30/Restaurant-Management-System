# Restaurant-Management-System
A Restaurant Management System is a software solution designed to manage daily restaurant operations efficiently. It helps in handling orders, billing, table reservations, inventory, and staff management. 
# Restaurant Management System (C)

A console-based **Restaurant Management System** written in **C language** using
**structures and file handling**.

The system allows an **Admin** to manage food items and **Customers** to
register, view menu items, and place orders.
